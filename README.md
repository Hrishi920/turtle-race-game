A beginner-friendly Python game built using the `turtle` graphics module, where the user bets on which turtle will win a race.

This project was created to practice Python fundamentals such as loops, conditionals, lists, and basic game logic.

---

## How it works
- Six turtles are placed at the starting line
- The user is asked to enter a colour as a bet
- Only valid turtle colors are accepted
- Each turtle moves forward by a random distance
- The first turtle to cross the finish line wins
- The program tells the user whether thet won or lost

---

## Features
- User input validation
- Visual feedback using turtle graphics
- Randomized race movement
- Clear win/lose result.
